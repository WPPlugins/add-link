Adds a sidebar widget to submit links to blogroll

== Description ==

This is a simple plugin that enables users to add a sidebar widget to submit links to the blogroll.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the add-link.zip file to a directory of your choice(preferably the wp-content/plugins/widgets folder)

2. Unzip the add-link.zip file into the wordpress plugins/ directory: 'wp-content/plugins/'

3. Activate the 'Add Link' plugin from the Plugins' menu in WordPress

4. Access the widgets menu and add the add-link widget to your pages.
